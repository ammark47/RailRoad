To start the program:

1. Enter these commands in this cd of this folder:
  `pip install flask`
  `pip install xlrd`
  `pip install sqlite3`
  `pip install datetime`

2.  install the sql database(this takes a long time):
  `py sql.py`

3. run
  `py app.py`

4. open this link in your favorite browser:
  http://localhost:8000/
